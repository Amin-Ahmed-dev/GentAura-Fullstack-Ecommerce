# 👔 GentAura | Luxury Formal Menswear
A professional Full-Stack E-commerce platform for high-end men's fashion.

## 🚀 Features
- **Frontend:** React (Vite) with Context API for state management.
- **Backend:** Node.js & Express API.
- **Database:** MySQL with JSON data types for flexible product attributes.
- **Security:** JWT Authentication and Bcrypt password hashing.
- **UI/UX:** Dark luxury theme, responsive grid, and custom skeleton loaders.

## 🛠️ Tech Stack
- **Frontend:** React, Axios, Bootstrap, Lucide Icons, SweetAlert2.
- **Backend:** Express, MySQL2, JWT, Dotenv.

## 📦 Installation & Setup
1. **Database:** Import the `backend/database.sql` file into your local PHPMyAdmin/MySQL.
2. **Backend:**
   - Go to `/backend`
   - Run `npm install`
   - Create a `.env` file with your DB credentials.
   - Run `npm run dev`
3. **Frontend:**
   - Go to `/frontend`
   - Run `npm install`
   - Run `npm run dev`
